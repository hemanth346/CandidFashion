About CandidFashion

Being able to recognize apparel products from pictures could enhance the shopping experience for consumers.
This platform aims to makes the apparel items, in given fashion image, available as separate image - ready to be able to search and buy online.
Reverse Image search currently has to be done manually using platforms like Lykdat but can be integrated as an api service.
(Lykdat doesn't have an open APi, but has a twitter bot that can be used)
